# Pathlyft

Lightweight AI-assisted job applications UI.

## Quickstart

Requirements: Node.js (LTS) and npm.

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Open http://localhost:5173 (or the URL printed by Vite)

## What technologies are used?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build and preview locally:

```bash
npm run build
npm run preview
```

For production hosting, deploy the `dist/` output to your preferred static host (Vercel, Netlify, Netlify, Cloudflare Pages, S3 + CloudFront, etc.).
